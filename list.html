리스트 만들기

리스트를 나열하는 방법은 3가지가 있다.

  • 순서가 있는 리스트(ordered list) - <ol></ol>
  • 순서가 없는 리스트(unordered list) - <ul></ul>
  • 정의 리스트(definition list) - <dl></dl>

각 아이템은 <li> 태그를 사용하며, </li>는 생략 가능하다.

ol 태그

  1. 순서가 있는 리스트1
  2. 순서가 있는 리스트2
  3. 순서가 있는 리스트3

type 속성을 이용하여 마커 종류를 설정할 수 있다.(1, A, a, I, i)

start 속성을 이용하여 마커의 시작값을 설정할 수 있다.(숫자값)


ul 태그

  • 순서가 없는 리스트1
  • 순서가 없는 리스트2
  • 순서가 없는 리스트3

중첩 리스트

리스트는 중첩도 가능하다.

  • 좋아하는 것
    1. 음식
    2. 노래
  • 싫어하는 것
    1. 공부
    2. 벌레
  • 취미

dl 태그

정의 용어
정의 설명

dl 태그는 아이템으로 dt(용어), dd(설명) 태그를 사용한다.

dt, dd 모두 닫는 태그는 생략 가능하다.
