input 태그의 속성

  • type: 인풋 속성의 종류
  • name: 요소 이름
  • maxlength: 입력할 수 있는 문자 최대 개수
  • size: 입력 창의 크기
  • value: 버튼에서는 버튼 값, 입력창은 기본값

textarea 태그의 속성

  • cols: 영어 기준 입력 창 가로 크기
  • rows: 영어 기준 입력 창 세로 크기
  • wrap: 자동 줄바꿈 처리 지정(OFF/HARD/SOFT)

datalist 태그

나라:

input 태그의 list 속성과 함께 사용한다.

선택 관련 태그(checkbox/radio/select)


checkbox

선택 1 선택 2 선택 3

checked 속성을 이용해서 기본적으로 체크될 항복 설정 가능


radio

선택 1 선택 2 선택 3

name 속성을 이용해서 같은 값을 가진 라디오버튼은 하나만 선택 가능


select

select 태그 안에 option 태그를 이용하여 요소 나열이 가능하다.
option 태그 안에 selected 속성을 이용해서 기본 선택값 지정 가능

datalist는 리스트 안에서 선택 하거나 직접 입력이 가능하나 select는 옵션 안에서 선택만 가능하다.
