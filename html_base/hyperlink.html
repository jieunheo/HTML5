하이퍼링크

HTML 페이지에서 다른 HTML 페이지를 연결하는 것을 하이퍼링크라 한다.
하이퍼링크는 텍스트나 이미지에 다른 웹 페이지의 주소를 달아서 만든다.
하이퍼링크에 따라 다른 웹 페이지로 이동하는 것을 항해 혹은 내비게이션이라고 부른다.
웹 페이지 경로 작성을 위해서는 절대경로와 상대경로 두 가지 방법이 있다.

a 태그의 href 속성을 이용하여 하이퍼링크를 만들 수 있다.

a 태그에서 사용 가능한 속성

  • href: 이동할 HTML 페이지의 URL 또는 HTML 페이지 내 앵커 이름
  • target: 링크에 연결된 HTML 페이지가 출력될 윈도우 이름 지정
  • download: 링크가 클릭되면 href에 지정된 파일이 다운로드
네이버 구글

링크의 텍스트는 사용 과정에 따라 색이 3가지로 분류된다.

  • 처음 링크(standard link): 밑줄과 함께 blue
  • 방문이 이루어진 링크(visited link): purple
  • 마우스로 링크를 누르고 있는 동안(active link): red


target 속성을 이용하여 링크가 열리는 위치를 정할 수 있다.

  • _blank: 새 창
  • _self: 현재 창. 기본값
  • _parent: 현재 창을 열게 한 창. 없다면 현재 창
  • _child: 새 창
  • _top: 제일 먼저 열린 창


id 속성으로 앵커 만들기

name 속성을 가지고도 가능하다.
