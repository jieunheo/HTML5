# HTML5
HTML5 공부

##HTML이란?
*HyperText Markup Language의 약자
*웹을 이루는 가장 기초적인 구성 요소로, 웹 콘텐츠의 의미와 구조를 정의할 때 사용
*CSS와 Javascript를 함께 사용

*"태그"를 사용해서 문서의 다른 텍스트와 구분하며, <태그></태그>와 같이 시작태그와 끝태그가 한 쌍으로 이루어짐
