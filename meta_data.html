메타 데이터

메타 데이터란 데이터를 설명하는 데이터이다.
HTML에서도 HTML 페이지에 대한 메타 정보를 표현하는 여러 태그가 있다.

<base>, <link>, <script>,
<style>, <title>, <meta>

base 태그

base 태그는 웹 페이지들이 담겨 있는 기본 URL과 웹 페이지가 출력될 윈도우를 지정하기 위해 사용된다.

base 태그에 href를 이용하여 URL을 지정한다.

a태그에 base 태그가 들어간 파일의 이름을 href를 이용하여 넣어주면, a태그 클릭시 base 태그에 적힌 페이지로 이동한다.
